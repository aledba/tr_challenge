ROOT CAUSE ANALYSIS

1. Limited compute: Apple MPS, not GPU cluster. Batch size=8 was necessary.

2. pos_weight overcompensation: Max weight 50x pushed model toward rare classes, hurting precision.

3. Only 5 epochs: Best at epoch 4 but F1 still climbing. Needed more training time.

4. Summarization loss: 33% of docs lost information through LED compression.

5. No hyperparameter tuning: Single configuration run, no grid search.
